# Sub-Level 30.1

### **Description**

This level covers the same concepts with simpler clock readings focused on half-hour times. Students observe clocks and write the correct time.

### **Sections**

#### **Section A: Write the Time**

* Observe and record time.

#### **Section B: Match Clock and Time**

* Match clock faces with half-hour times.

### **Examples**

🕜 Clock shows **1:30**

Write the time.

🕟 Clock shows **4:30**

Write the time.

🕢 Clock shows **7:30**

Write the time.

🕥 Clock shows **10:30**

Write the time.
