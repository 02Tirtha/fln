# Main Level 30.0

### **Description**

This level assesses mastery of reading analog clocks showing quarter-hour and mixed times. Students observe clock faces and write the correct time independently.

### **Sections**

#### **Section A: Read the Clock**

* Observe the clock and write the time.

#### **Section B: Match Time and Clock**

* Match clock images to the correct time.

### **Examples**

🕐 Clock shows **1:10**

Write the time.

🕑 Clock shows **2:15**

Write the time.

🕓 Clock shows **4:45**

Write the time.

🕘 Clock shows **9:20**

Write the time.
