# Sub-Level 30.2

### **Description**

This level provides maximum support by introducing full-hour times only. 

Students identify clocks where the minute hand points to 12 and write the corresponding hour.

### **Sections**

#### **Section A: Write the Hour**

* Observe and write the time.

#### **Section B: Match Clock and Number**

* Match clocks to the correct hour.

### **Examples**

🕐 Clock shows **1:00**

Write the time.

🕔 Clock shows **5:00**

Write the time.

🕒 Clock shows **3:00**

Write the time.

🕗 Clock shows **8:00**

Write the time.
