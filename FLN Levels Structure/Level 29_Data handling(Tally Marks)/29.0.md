# Main Level 29.0

### **Description**

This level assesses mastery of tally marks and data interpretation. Students create and read tally charts involving multiple categories and compare quantities using the recorded data.

### **Sections**

#### **Section A: Convert Quantity to Tally**

* Given a number (e.g., 17), represent it using tally marks.

#### **Section B: Read Tally Marks**

* Determine the quantity represented by tally marks.

#### **Section C: Data Interpretation**

* Answer visual questions using tally charts.
