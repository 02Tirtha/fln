# Sub-Level 29.1

### **Description**

This level covers the same concepts with smaller quantities and simpler tally representations. Students practice reading and creating tally marks with fewer categories.

### **Sections**

#### **Section A: Number → Tally**

* Represent numbers up to 20–30 using tally marks.

#### **Section B: Tally → Number**

* Count tally marks and write the number.

#### **Section D: Compare Categories**

* Identify more and less.
